# Minimalist IME (极简输入法) 🚀

这是一款基于 Android 原生 API 开发的轻量级拼音输入法，具有极高的响应速度和极小的内存占用。

## ✨ 功能特性
- **零内存占用**：正如名字 "No Memory Required" 所言，极其精简。
- **动态词库**：支持从外部 assets 加载并导入 SQLite 数据库。
- **滑动候选栏**：流畅的 HorizontalScrollView 交互体验。
